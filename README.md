# narayan-pharmacy-task

Prescription Entry & Drug Interaction Checker built with Django and Next.js.

This take-home assessment is intentionally scoped to two screens only:

1. Prescription Entry Form
2. Prescriptions List and Detail View

See `CLAUDE.md` for architecture, local development commands, coding conventions, and scope constraints.

## Quick Start

```bash
cp .env.example .env
./setup.sh
```

Then run the backend and frontend in separate terminals:

```bash
cd backend
python manage.py runserver
```

```bash
cd frontend
npm run dev
```
